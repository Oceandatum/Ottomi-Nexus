## 🚀 核心优势

### 1. 📦 单包一体化部署，真正简单易用

Ottomi Nexus 采用单包交付，一键安装即可上线，无需复杂环境配置与二次开发。

内置数据接入、智能治理、数仓建模、数据质检、资产管控、BI 分析、权限审批全链路能力，让企业从 “多系统拼凑” 升级为**一个平台搞定全域数据问题**。

### 2. 🤖 AI 辅助自动化治理，大幅降低人力成本

平台通过 AI 辅助实现数据全流程智能化：

- 自动编目、自动归集源端资产
- CDC 实时同步与批量数据一体化处理
- 自动化数据质检（符合 DAMA 标准）
- 智能推荐数据标准、指标、维度模型

将传统人力密集型的数据治理，变为**可视化、点选式、低成本**的高效操作。

### 3. 🔒 安全可控的数据空间，分类分级自动扫描 + 分级共享

构建企业级安全合规的数据治理体系：

- 分类分级自动识别与扫描
- 全链路数据血缘追踪
- 分级共享机制，数据按需安全开放
- 双沙盒开发 / 生产环境隔离
- 防篡改日志，确保操作可审计、可追溯

让数据真正做到**可控、可管、可共享、可合规**。

### 4. 📊 可视化数仓建模，轻松构建多维 Cube 模型

提供完整企业级数仓建模能力：

- 可视化维度表、事实表构建
- 拖拽式多维 Cube 设计
- 可视化配置生成指标体系
- 从原始数据到加工资产的全流程可视

降低数仓建设门槛，让业务人员也能轻松参与数据建模。

### 5. 🏛️ 全链路数据资产运营，让数据成为可运营资产

数据资产不再只包含数据表，而是统一管理：

- 原始数据、加工过程、资产发布
- 指标体系、API 服务、业务流程
- 数据申请、审批、订阅、授权全流程

实现数据资产**可盘点、可追溯、可申请、可审批、可运营**。

### 6. 🧩 轻量级一体化操作，上手快、学习成本低

全部功能在统一 Web 界面完成，基本无需编写脚本：

- 批量点选式数据集成，无需逐表配置
- 内置消息通知（站内、邮件、企业微信）
- 细粒度权限管控（数据源、表、列、行级权限）
- 内置流程审批引擎，数据申请自动流转

真正实现**轻量、简单、一体化、低成本**的数据治理体验。

## ⚡ 快速上手

Ottomi Nexus 部署包体积较大，采用 **在线拉取镜像 + 离线私有化部署** 双模式，支持有网 / 无网环境快速部署，约 15 分钟完成基础搭建。

### 在线部署（约15分钟）

在具备网络条件的服务器，直接从官方镜像仓库拉取最新镜像资源，完成容器化一键安装，自动依赖补齐与环境配置，快速启动平台。

### 离线部署（约15分钟）

首先在有网服务器拉取官方镜像并导出为离线安装包；将安装包拷贝至内网 / 无网环境后，导入镜像并执行本地安装，全程无外部网络依赖、无数据外流。

部署完成后内置完整 Demo 项目与 Demo 数据空间，开箱即用，无需额外准备测试数据，可直接体验平台全流程功能。



## 🖥️ 系统环境要求

- OS：支持主流 Linux 发行版本（基于 Debian / RedHat，包括国产操作系统）
- 服务器架构：目前仅支持X86_64（后续会陆续开放aarch64架构）
- 最低资源要求：8C16G

## 🛠️ 技术栈

Ottomi Nexus 构建于云原生架构之上，深度集成主流开源生态与异构数据能力，打造高效、稳定、合规的数据治理底座：

- 云原生与分布式架构

  支持容器化部署、弹性伸缩与分布式并行计算，从容应对海量多模态数据处理需求。

- 内置开源生态组件

  深度整合 

  SeaTunnel、DolphinScheduler、DataHub、DataEase、MinIO等开源项目，内置全链路数据接入、调度、元数据管理与 BI 可视化能力，大幅降低二次开发成本。

- 多源异构数据兼容

  全面支持 

  MySQL、PostgreSQL、Oracle、DB2、达梦、人大金仓、OceanBase、Clickhouse等主流关系型数据库，同时兼容 Kafka、Hive等大数据生态组件，实现结构化、非结构化及实时流数据的统一接入与自动映射。且用户可以自行选择系统兼容的数据库类型，作为数仓使用。

- 多模态数据治理

  覆盖关系型数据、日志文件、音视频等多类型数据，打破数据孤岛，实现全域数据统一管理。

- 合规适配体系

  深度适配《数据安全法》《个人信息保护法》及 GDPR 相关要求，内置

  分类分级自动扫描、分级共享、双沙盒隔离、防篡改日志

  等能力，助力企业构建合规可控的数据流通体系。

- 轻量级一体化界面

  基于 Web 统一界面交付，无需复杂脚本，支持全员快速上手。
