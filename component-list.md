# 数据开发组件清单

本文档汇总了Ottomi Nexus平台内置的全量数据开发组件，按功能场景分为实时输入、实时输出、离线输入、离线输出、数据转换、离线脚本、离线数据运算、离线多表同步、离线数据融合九大类别，所有组件均支持可视化拖拽配置，无需编码即可完成全链路数据开发。随着软件的迭代将会增加更多的组件，并且希望支持用户自定义组件和函数。

## 1. 实时输入组件（适用于实时开发和实时质检）
| 组件名称 | 功能说明 |
|---------|---------|
| Kafka输入 | 用于从Kafka消息队列中读取实时数据流，作为实时任务的输入源 |
| MysqlCDC | 基于MySQL的CDC（变更数据捕获）能力，实时读取MySQL数据库的增量变更数据 |
| OracleCDC | 基于Oracle的CDC能力，实时读取Oracle数据库的增量变更数据 |
| SQLServerCDC | 基于SQL Server的CDC能力，实时读取SQL Server数据库的增量变更数据 |
| MongoDbCDC | 基于MongoDB的变更流能力，实时读取MongoDB数据库的增量变更数据 |
| PostgresqlCDC | 基于PostgreSQL的逻辑复制能力，实时读取PostgreSQL数据库的增量变更数据 |
| EventStore输入 | 从EventStore事件存储中读取实时事件流数据，作为实时任务的输入源 |

---

## 2. 实时输出组件
| 组件名称 | 功能说明 |
|---------|---------|
| 实时单表输出 | 将实时处理后的结果数据，实时写入到目标数据库的单表中 |
| 实时Starrocks输出 | 将实时处理后的结果数据，实时写入到Starrocks分析型数据库中 |
| 实时Kafka输出 | 将实时处理后的结果数据，实时写入到Kafka消息队列中，供下游系统消费 |

---

## 3. 离线输入组件
| 组件名称 | 功能说明 |
|---------|---------|
| 单表输入 | 从各类关系型数据库（MySQL、Oracle、SQL Server、人大金仓、达梦等）的单表中读取离线数据 |
| API输入 | 支持通过Http、Https、SOAP协议读取API接口数据，支持在请求头部添加静态/动态Token验证 |
| MongoDb输入 | 从MongoDB数据库中读取离线数据 |
| Starrocks输入 | 从Starrocks分析型数据库中读取离线数据 |
| Excel输入 | 读取Excel格式的离线文件数据，支持.xls、.xlsx格式 |
| Csv输入 | 读取CSV格式的离线文件数据，支持自定义分隔符、编码格式 |
| Xml文件输入 | 读取XML格式的离线文件数据，支持XPath路径提取 |
| 文本文件输入 | 读取普通文本格式的离线文件数据，支持自定义行分隔符、编码格式 |
| S3文件输入 | 读取S3协议兼容的对象存储中的离线文件数据 |
| Json文件输入 | 读取JSON格式的离线文件数据，支持JsonPath路径提取 |
| 逻辑表输入 | 读取平台内定义的逻辑表数据，实现跨源数据的统一读取 |
| FTP输入 | 从FTP服务器中读取离线文件数据 |
| SFTP输入 | 从SFTP安全文件传输服务器中读取离线文件数据 |
| rabbitMQ输入 | 从RabbitMQ消息队列中读取离线数据 |

## 5. 数据转换组件（实时和离线通用）
| 组件名称 | 功能说明 |
|---------|---------|
| 异常值检测 | 检查字段数据是否符合预设的质检规则，可分别输出符合规则/不符合规则的数据集 |
| 唯一Id生成 | 为数据行生成全局唯一的ID标识，支持雪花ID、UUID等多种生成规则 |
| 列转行 | 将一列或多列的结构化数据，按指定规则拆分为多行数据 |
| NULL值替换 | 将数据集中的空值（NULL）按指定规则替换为默认值、填充值 |
| 数据过滤 | 按指定的条件规则，对数据集进行行级过滤，保留符合条件的数据行 |
| 值替换 | 按指定的匹配规则，对数据集中的字段值进行批量替换 |
| 字符串修剪 | 去除字符串字段的首尾空格、指定字符，实现字符串格式化 |
| 大小写转换 | 将字符串字段的内容统一转换为大写或小写格式 |
| 字符串拆分 | 将指定字符串字段按特殊字符、分隔符进行拆分，生成新的字段列 |
| 字符串拼接 | 将多个字段的内容按指定规则拼接为一个新的字符串字段 |
| 字符串剪切 | 对指定字段的内容进行截取、剪切，替换原字段或生成新的字段 |
| 字段过滤 | 从数据集中去除指定的字段列，仅保留需要的字段 |
| 字段名称映射 | 按指定的映射规则，替换数据集的字段名称，实现字段重命名 |
| Java高级转换 | 支持在线编写Java代码，对数据集进行自定义的复杂处理与转换 |
| JsonPath提取 | 从JSON类型的字段中，按JsonPath规则提取指定内容，生成新的字段 |
| 函数计算 | 对指定列的数据，通过预设的数值、字符、时间函数进行转换，生成新的计算列 |
| 数据解密 | 对加密的字段数据，按指定的加密算法与密钥进行解密处理 |
| 数据加密 | 对敏感字段数据，按指定的加密算法与密钥进行加密处理 |
| 数据脱敏 | 对敏感字段数据按指定规则进行脱敏处理，支持掩码、替换、部分隐藏等多种脱敏方式 |

## 6. 离线脚本组件
| 组件名称 | 功能说明 |
|---------|---------|
| 脚本管理 | 支持自定义脚本的上传、在线编辑、删除、版本管理，统一管理各类离线脚本 |
| SQL脚本 | 支持在线编写、执行自定义SQL语句，实现数据查询、处理、更新等操作 |
| shell脚本 | 支持编写、执行shell脚本，实现系统级操作、文件处理、任务调度等功能 |
| Python脚本 | 支持编写、执行Python脚本，实现复杂数据处理、模型调用、自定义逻辑开发 |
| flink脚本 | 支持编写、执行Flink处理脚本，实现离线/实时数据处理 |
| MR脚本 | 支持编写、执行MapReduce脚本，实现大数据集的分布式离线计算 |
| FlinkSQL脚本 | 支持编写、执行FlinkSQL语句，实现流批一体的数据处理与分析 |
| HQL脚本 | 支持编写、执行Hive SQL语句，实现Hive数据仓库的查询、处理、建模 |
| Datax脚本 | 支持编写、执行Datax同步脚本，实现异构数据源之间的离线数据同步 |
| sqoop脚本 | 支持编写、执行sqoop脚本，实现Hadoop与关系型数据库之间的数据传输 |
| flinkjar | 支持上传、执行自定义的Flink Jar包，实现复杂的Flink任务开发与执行 |

## 7. 离线数据运算组件
| 组件名称 | 功能说明 |
|---------|---------|
| 聚合运算 | 对数据集按指定的分组字段进行聚合计算，支持SUM、COUNT、AVG、MAX、MIN等多种聚合函数 |
| 数据去重 | 按指定的字段或全字段，对数据集进行去重处理，保留唯一的数据行 |
| 数据排序 | 按指定的字段与排序规则（升序/降序），对数据集进行全量排序 |

## 8. 离线多表同步组件
| 组件名称 | 功能说明 |
|---------|---------|
| 多表同步 | 支持多个源表到目标表的批量离线数据同步，可配置同步规则、字段映射、过滤条件，实现多表的统一同步调度 |

|---------|---------|
