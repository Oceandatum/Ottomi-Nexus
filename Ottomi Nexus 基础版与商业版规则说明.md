## 一 整体产品模式

Ottomi Nexus 采用**基础版永久免费 + 商业版两档逐级购买解锁**的产品模式。后续将逐步开源数据应用部分模块（如智能问数、多模态搜索、主数据管理以及智能中心调用系统 API/MCP调用示例与开发规范等），核心技术模块保持闭源。

- 基础版：面向中小微企业、小团队、个人开发者，永久免费下载，支持私有化部署使用；
- 商业版：分为**高级版、集团版**两档，**必须先购买高级版，方可购买集团版**，实行永久授权、年度服务分离机制；
- 所有商业合作、授权解锁、服务选购，均可对接 Oceandatum 官方销售团队沟通确认。

## 二 各版本资源限制明细

### （一）基础版（永久免费版）

适用对象：中小微企业、初创团队、个人开发者，无需企业认证即可使用。

核心承诺：**功能无阉割，仅做资源量化限制**，产品核心功能完全开放。

|    资源分类    |                      具体限制                       |
| :------------: | :-------------------------------------------------: |
|  并行计算节点  |      节点数≤1（最小配置：8 核 CPU + 16G 内存）      |
| 人员与协作规模 |     注册用户数≤8；项目空间数≤30；数据源数量≤12      |
|  数据处理规模  | 数据质量检测模型 + 数据开发模型≤500；指标模型数≤100 |
| 调度与服务规模 |    已配置调度任务数≤300；数据服务 / API 数量≤200    |

服务说明：默认无官方技术支持，不开放 Case 提报权限；可通过 GitHub 开源社区文档、问答区获取通用问题解答，如需官方技术支持可选购对应年度服务。

### （二）高级版（商业版）

适用场景：中型企业、多业务线、常规多租户数据治理场景。

核心权益：解锁基础版资源限制，固定资源上限，配套专属商业服务。

|    资源分类    |                       具体限制                       |
| :------------: | :--------------------------------------------------: |
|  并行计算节点  |                 ≥2 节点，可按需扩容                  |
| 人员与协作规模 |     注册用户数≤40；项目空间数≤150；数据源数量≤60     |
|  数据处理规模  | 数据质量检测模型 + 数据开发模型≤2500；指标模型数≤500 |
| 调度与服务规模 |   已配置调度任务数≤1500；数据服务 / API 数量≤1000    |

解锁方式：通过软件管理中心生成 MAC 地址等环境码，官方根据环境码生成激活码，输入激活码解锁对应资源上限。

### （三）集团版（商业版）

适用场景：集团型企业、多分子公司、超大规模多租户数据中台场景。

核心权益：**叠加高级版全部能力，全维度无资源使用上限**。

- 并行计算节点：无数量上限，免费无限扩容。
- 人员、项目、数据源、各类模型、调度任务、数据服务 / API：全部无数量限制。

解锁方式：已购买高级版并完成授权后，通过官方激活升级，替换全域无限制 License，保留原有配置与数据。

### （二）Case 类型与计次规则

仅购买官方年度技术支持服务后，可开放 Case 提报权限，Case 按以下标准分级计次：

| Case 类型 |                           界定标准                           | 单次计次 |
| :-------: | :----------------------------------------------------------: | :------: |
| 轻量 Case | 功能用法询问、界面操作指导、配置项解释、报表 / 字段含义说明，单次沟通 30 分钟内可解决 |  0.5 次  |
| 常规 Case | 故障排查、数据问题定位、接口对接调试、复杂业务配置，单次处理时长超 30 分钟 |   1 次   |

### （三）上门服务与远程 Case 折算规则

客户可使用年度剩余有效 Case 次数，兑换单日上门支持服务，不同区域折算标准如下：

- 华东地区：1 天上门服务 = 3 次远程 Case
- 华北、华南地区：1 天上门服务 = 4 次远程 Case
- 西南、东北地区：1 天上门服务 = 5 次远程 Case
- 西北（含新疆）地区：1 天上门服务 = 6 次远程 Case

### （四）不计入 Case 次数的服务场景

- 产品原生 Bug、程序异常、系统崩溃、安全漏洞补丁及紧急更新推送；
- 官方统一公告、公共文档说明、群发通知类无一对一人工服务内容；
- 客户自主查阅文档、自助操作、自行排查，无工程师介入协助。

## 四 License 授权与解锁规则

1. 基础版：安装自动生成通用 License，默认锁定基础版资源上限，无设备绑定，无专属服务权限；
2. 高级版：完成商业授权办理后，发放设备专属 License，与 MAC 地址唯一绑定，开通对应服务与 Case 权限；
3. 集团版：完成升级授权后，替换全域无限制 License，自动解锁全部资源限制，同步升级服务档位；
4. 高级版新增计算节点，同步更新 License 完成扩容；集团版节点扩容无需重复授权；
5. 商业版永久授权长期有效，主线版本迭代功能可免费升级，不受服务续费状态影响；
6. 年度服务到期未续费，仅关闭人工技术支持与 Case 提报权限，不锁定已解锁软件资源。

## 五 定制化增值服务

全版本客户均可按需选购定制化增值服务，此类服务为独立项目，不占用年度 Case 次数，具体方案需对接官方销售洽谈：

1. 现场实施类：驻场部署、环境调试、落地培训、本地化适配；
2. 专属运维类：长期驻场、7×24 应急保障、定期巡检、专属 SLA 定制；
3. 商务支撑类：招投标资料提供、验收配合、资质授权、商务答辩；
4. 定制开发类：功能改造、第三方系统集成、个性化接口开发；
5. 咨询规划类：数据中台顶层设计、行业数据模型搭建、全域数据治理方案。

## 六 对外合作通用规则

1. 商业版授权、服务选购、渠道转售等相关合作细节，统一由官方销售团队对接，不对外公开具体合作报价；
2. 高级版、集团版实行逐级购买刚性规则，无高级版授权，无法直接办理集团版授权；
3. 基础版资源触达上限后，需升级商业版方可解除限制，保障系统稳定运行；
4. 本规则为对外通用标准，特殊行业、大型项目可按需定制专属合作方案；
5. 产品规则随版本迭代优化，官方将及时同步最新内容，最终解释权归 Oceandatum 所有。

## 七 补充声明

1. 商业版永久授权仅针对产品资源上限解锁与标准产品使用权，不包含定制化开发、现场实施等增值服务；
2. 严禁破解、篡改 License 授权，严禁非授权商用、二次分发，违者将追究相关责任；
3. 本规则未提及内容，以双方正式签订的合作协议为准。
