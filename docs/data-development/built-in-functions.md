# 内置函数说明
本文档汇总了Ottomi Nexus平台内置的全量数据开发函数，按功能场景分为数值函数、字符函数、系统函数、时间函数四大类，所有函数均支持离线/实时数据处理场景，可直接在数据开发画布、SQL脚本中调用。随着软件的迭代将会希望支持用户自定义函数。

## 1. 数值函数
| 函数名称 | 功能说明 |
|---------|---------|
| ABS | 计算并返回数值的绝对值 |
| ACOS | 计算并返回数值的反余弦值 |
| ASIN | 计算并返回数值的反正弦值 |
| ATAN | 计算并返回数值的反正切值 |
| CEIL | 返回大于或等于输入参数的最小整数值（向上取整） |
| COS | 计算并返回数值的三角余弦值 |
| COSH | 计算并返回数值的双曲余弦值 |
| COT | 计算并返回数值的三角余切值 |
| EXP | 计算并返回自然常数e的指定次幂 |
| FLOOR | 返回小于或等于输入参数的最大整数值（向下取整） |
| LN | 计算并返回数值的自然对数（以e为底） |
| LOG | 计算并返回数值的对数（可指定底数，默认以10为底） |
| LOG10 | 计算并返回数值的以10为底的常用对数 |
| MOD | 计算并返回两个数值相除后的余数（取模运算） |
| PI | 返回圆周率π的常量值 |
| POWER | 计算并返回数值的指定次幂 |
| RADIANS | 将角度值转换为弧度值 |
| RAND / RANDOM | 生成并返回0到1之间的随机浮点数 |
| ROUND | 将数值四舍五入到指定的小数位数 |
| SIGN | 返回数值的符号：正数返回1，负数返回-1，0返回0 |
| SIN | 计算并返回数值的三角正弦值 |
| SINH | 计算并返回数值的双曲正弦值 |
| SQRT | 计算并返回数值的平方根 |
| TAN | 计算并返回数值的三角正切值 |
| TANH | 计算并返回数值的双曲正切值 |
| TRUNC | 将数值截断到指定的小数位数（不做四舍五入） |
| UUID_NUMBER | 生成随机的长整型唯一ID |

## 2. 字符函数
| 函数名称 | 功能说明 |
|---------|---------|
| ASCII | 返回字符串中第一个字符的ASCII编码值 |
| BIT_LENGTH | 返回二进制字符串中的总位数 |
| CHAR | 将ASCII编码值转换为对应的字符 |
| CHAR_LENGTH | 返回字符串中的字符总数（支持中文等多字节字符） |
| CONCAT | 将多个字符串按顺序拼接为一个字符串 |
| CONCAT_WS | 使用指定的分隔符，将多个字符串拼接为一个字符串 |
| HEXTORAW | 将十六进制格式的字符串转换为RAW二进制数据 |
| INSERT | 在原始字符串的指定起始位置插入新的字符串，可覆盖指定长度的原内容 |
| LEFT | 从字符串的左侧开始，返回指定长度的子字符串 |
| LOWER | 将字符串中的所有大写字母转换为小写字母 |
| LPAD | 从字符串的左侧填充指定字符，使字符串达到指定的总长度；若原字符串长度超过指定值，则从右侧截断 |
| LTRIM | 去除字符串左侧的所有空格或指定的前导字符 |
| OCTET_LENGTH | 返回二进制字符串中的总字节数 |
| POSITION | 返回指定子字符串在原始字符串中首次出现的位置 |
| RAWTOHEX | 将RAW二进制数据转换为十六进制格式的字符串 |
| REGEXP_LIKE | 判断字符串是否与指定的正则表达式匹配，返回布尔值 |
| REGEXP_REPLACE | 将字符串中与正则表达式匹配的所有子串，替换为指定的新字符串 |
| REGEXP_SUBSTR | 提取字符串中与正则表达式匹配的子字符串 |
| REPEAT | 将字符串重复指定的次数，返回新的字符串 |
| REPLACE | 将字符串中所有匹配的搜索子串，替换为指定的新字符串 |
| RIGHT | 从字符串的右侧开始，返回指定长度的子字符串 |
| RPAD | 从字符串的右侧填充指定字符，使字符串达到指定的总长度；若原字符串长度超过指定值，则从左侧截断 |
| RTRIM | 去除字符串右侧的所有空格或指定的尾随字符 |
| SOUNDEX | 返回表示字符串发音特征的四字符编码，用于模糊语音匹配 |
| SPACE | 返回由指定数量的空格组成的字符串 |
| SUBSTR | 从字符串的指定起始位置开始，返回指定长度的子字符串 |
| TRIM | 去除字符串两侧的所有空格或指定的首尾字符 |
| UPPER | 将字符串中的所有小写字母转换为大写字母 |

## 3. 系统函数
| 函数名称 | 功能说明 |
|---------|---------|
| ARRAY | 生成一个数组类型的结果集 |
| CASE WHEN | 实现多条件分支判断，根据不同的条件返回对应的值 |
| CAST | 将一个值的数据类型强制转换为另一种指定的数据类型 |
| COALESCE | 返回参数列表中第一个不为NULL的值，若所有参数均为NULL则返回NULL |
| IFNULL | 若第一个参数不为NULL则返回第一个参数，否则返回第二个参数（NULL值替换） |
| LATERAL VIEW | 用于将数组、Map等复杂类型数据拆分为多行数据，实现列转行 |
| NULLIF | 若两个参数的值相等，则返回NULL，否则返回第一个参数的值 |
| UUID | 生成通用唯一识别码（UUID）字符串 |

## 4. 时间函数
| 函数名称 | 功能说明 |
|---------|---------|
| CURRENT_DATE | 返回当前系统的日期值（仅日期，无时间部分） |
| CURRENT_TIME | 返回当前系统的时间值（仅时间，无日期部分） |
| CURRENT_TIMESTAMP | 返回当前系统带时区的完整时间戳（日期+时间+时区） |
| DATEADD | 对日期/时间值进行加减运算，返回计算后的新日期/时间值 |
| DATEDIFF | 计算两个日期/时间值之间的时间差，可指定年、月、日、时、分、秒等单位 |
| DATE_TRUNC | 将日期/时间值按指定的时间单位（年、月、日、时等）进行截断，返回该单位的起始值 |
| DAYNAME | 返回日期值对应的星期名称（如Monday、Tuesday） |
| DAY_OF_MONTH | 返回日期值在当月中的天数（1-31） |
| DAY_OF_WEEK | 返回日期值在当周的星期序号（1-7，1代表星期一，7代表星期日） |
| DAY_OF_YEAR | 返回日期值在当年中的天数（1-366） |
| EXTRACT | 从日期/时间值中提取指定的时间部分（年、月、日、时、分、秒等） |
| FORMATDATETIME | 将日期、时间或时间戳值，按指定的格式字符串转换为文本格式 |
| HOUR | 从时间/时间戳值中提取小时部分（0-23） |
| MINUTE | 从时间/时间戳值中提取分钟部分（0-59） |
| MONTH | 从日期/时间戳值中提取月份部分（1-12） |
| MONTHNAME | 返回日期值对应的月份名称（如January、February） |
| PARSEDATETIME | 将文本格式的日期/时间字符串，按指定的格式解析为标准的日期/时间类型 |
| QUARTER | 从日期/时间戳值中提取季度部分（1-4） |
| WEEK | 从日期/时间戳值中提取当年的周序号（1-53） |
| YEAR | 从日期/时间戳值中提取年份部分 |
