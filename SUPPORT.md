# 产品支持说明

## 支持渠道与服务范围

本项目针对**基础版**与**商业版**用户，提供差异化技术支持服务，所有支持均通过官方指定渠道对接，不接受非正规渠道的咨询与服务请求。

### 一、基础版用户支持（永久免费）

1. 支持渠道：仅通过 GitHub Issues 提供公共社区答疑，仅处理通用功能使用、部署环境适配、公开性问题排查类咨询；
2. 不支持范围：不提供一对一专属技术指导、不处理定制化需求、不提供深度故障排查、不提供上门实施与培训服务；
3. 响应时效：社区问题将在工作日不定期回复，无固定响应时效承诺。

### 二、商业版用户支持（企业专属）

1. 专属支持渠道：商务对接邮箱 info@oceandatum.com；
2. 支持范围：包含产品部署调试、全功能使用指导、专属故障排查、性能优化、安全加固、多租户配置等全场景技术支持，可协商上门实施、团队培训、定制化功能对接服务；
3. 响应时效：工作日内提供快速响应，遵循企业级服务时效承诺；
4. 额外权益：优先获取版本更新、安全漏洞修复、专属方案定制等增值服务。

## 咨询须知

1. 提交问题时，请注明产品版本、部署环境、问题复现步骤、相关日志 / 截图，便于高效排查；
2. 涉及商业授权、付费升级、定制开发等需求，直接通过商务邮箱对接，请勿在公共社区发布；
3. 产品使用需严格遵循项目 LICENSE、NOTICE 文件相关约定，违规使用产品导致的问题，不在支持范围内。
